
## Installation

---

Use the package manager to install modules

```
npm install
```

Create a database file called 
```[project-database.db]```
 in the same folder as 
 ```[app.js]```
Initilaise it by running the SQL found in 
```/sql/project-db-initalise-final-submission.sql```
